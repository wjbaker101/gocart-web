# gocart-web

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Secret Properties Config

`common/config-secret.json`
```
{
    "tescoAPI": {
        "secretKey": "<secret_key>"
    },
    "backend": {
        "apiKey": "<api_key>"
    }
}
```
